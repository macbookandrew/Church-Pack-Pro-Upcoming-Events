== Description ==

A mini WordPress plugin based on [Church Pack Pro](https://www.wpforchurch.com/wordpess-products/church-pack-pro/) that adds support for six months’ worth of events rather than just the current month.


== Installation ==

1. Install and activate [Church Pack Pro](https://www.wpforchurch.com/wordpess-products/church-pack-pro/) first.
1. Upload this plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode `[upcoming_events]` in the page where you wish the six months of events to appear
